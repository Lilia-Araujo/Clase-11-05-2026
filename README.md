Este programa sirve para convertir unidades de manera sencilla.

Primero muestra dos opciones en pantalla para que el usuario elija qué quiere convertir. Después, el usuario escribe una opción usando el teclado.

Si elige la primera opción, el programa pide una temperatura en Celsius y la convierte a Fahrenheit. Si elige la segunda opción, pide una cantidad de metros y la convierte a kilómetros.

Al final, el programa muestra el resultado de la conversión en pantalla.

También tiene un pequeño control para evitar errores: si el usuario escribe una opción incorrecta, el programa avisa que la opción no es válida.
